本程序在源程序[彩虹外链](http://blog.cccyun.cn/post-406.html)的基础上进行修改而来

 * 修改文件列表可以直接点击文件名称查看；
 * 更改了UI色彩，调整了部分UI布局；
 * 增加用户协议导航按钮，增加后台管理导航按钮；
